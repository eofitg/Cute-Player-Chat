## 使用方法 - 聊天后缀
使用
~~~
/getsuff [playerName_1] ... [playerName_n]
~~~
来查询自己或者多个玩家的聊天后缀

使用
~~~
/setsuff <suffix> [playerName_1] ... [playerName_n]
~~~
来改动自己或者多个玩家的聊天后缀

使用
~~~
/delsuff [playerName_1] ... [playerName_n]
~~~
来删除自己或者多个玩家的聊天后缀

以后会增加更多对玩家聊天内容的修改方式，来实现将任意玩家的聊天信息萌化（?）的效果
